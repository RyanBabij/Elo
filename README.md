# Elo
An R script that calculates Elo ratings and generate a chart over time.

## Required R libraries
* RColorBrewer

## Example of usage
### Input
```
Adam,win,Timothy
Adam,win,Timothy
Timothy,win,Jacqui
Timothy,win,Jacqui
Timothy,win,Jacqui
Timothy,draw,Jacqui
Dominic,win,Amy
Dominic,win,Amy
Timothy,win,Dominic
Timothy,win,Jacqui
Timothy,win,Adam
Timothy,win,Adam
```
